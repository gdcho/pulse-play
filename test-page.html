Video Speed Extension Test Page

Test Instructions:

  1. Load this page in Chrome with the extension installed
  2. Open the extension popup and configure your hotkey (default: Space)
  3. Click "Save Settings" in the popup
  4. Play the video below
  5. Hold your configured hotkey to speed up the video
  6. Release the hotkey to return to normal speed

Troubleshooting:

  • Check the browser console (F12) for any error messages
  • Make sure the extension is enabled and has permissions
  • Try refreshing the page after changing settings
  • Test with different hotkeys if the default doesn't work
